# SAQR ERP Desktop App

تطبيق سطح مكتب مبني بـ Electron + React مع API محلي وSQLite.

## التشغيل محليًا

```bash
cd desktop-app
npm install
npm run dev
```

سيعمل التطبيق على واجهة Vite (`http://localhost:5173`) والـ API المحلي على `http://localhost:3001`.

## ملاحظات
- قاعدة البيانات تُنشأ تلقائيًا داخل مجلد بيانات المستخدم.
